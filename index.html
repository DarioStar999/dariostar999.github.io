Nike Client

Un fork di Cloud Client da Github

Basato su Forge

A differenza della maggior parte degli altri client, Nike Client è una mod Forge, che ti consente di usarla in combinazione con innumerevoli altre modifiche!

Open Source

Il Client è un progetto open source che permette a tutti di contribuire! Basta visitare la pagina Github e dare un'occhiata.

Visivi Eccezionali

Cloud Client è ricco di elementi visivi estremamente accattivanti, come interfacce sfocate e arrotondate. Il nostro client offre anche animazioni UI fluide, mai viste prima in un client PVP.

Con oltre 50 impostazioni diverse, inclusi Checkbox, Slider, ModePicker e ColorPicker, e altro ancora, non c'è bisogno di preoccuparsi della personalizzazione.
